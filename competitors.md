# 競品追蹤清單（設定檔）

> 這份檔案是給 Claude 每次執行分析時讀取的清單。
> **新增/移除競品、改網址，都改這裡，不用去改 Routine 的 prompt。**

---

## 資料架構說明（重要）

`tracking-data.json` 現在改為**以月份為單位**儲存，結構是：

```
{ "months": [ {month1的完整資料}, {month2的完整資料}, ... ] }
```

每次 Routine 執行時：
1. 先判斷本次掃描屬於哪個月份（YYYY-MM）
2. 檢查 `months` 陣列中是否已有該月份的物件
   - 若有：更新該月份物件的內容（同月內重複執行視為更新，不要重複新增）
   - 若無：在陣列**最後面**新增一個新月份物件
3. 每個月份物件包含四個區塊，缺一不可：
   - `trends`：本月保健食品產業趨勢/新知（3-5則）
   - `powerHeroAnalysis`：PowerHero當月優劣勢分析+建議
   - `indicators`：五大指標橫向比較（同既有架構）
   - `log`：本月變化紀錄

## 月份內容撰寫指引

### 本月趨勢與新知（trends）
每則包含 `title`（簡短標題）與 `summary`（2-3句說明）。
查找方向：
- 搜尋「男性保健食品 趨勢 [當月年月]」「瑪卡 精胺酸 市場 新聞」
- 留意主要保健食品評測網站/部落客的最新評比文章中提到的產業觀察
- 留意法規異動（衛福部公告）、新原料專利、消費者輿論風向轉變
- 3-5則即可，不用多，但每則都要是「本月新出現或值得注意」的內容，不要寫恆常不變的衛教知識

### PowerHero 本月優劣勢分析（powerHeroAnalysis）
這是針對**我方品牌**的分析，依據是「PowerHero 跟本月查到的競品資料相比，現在站在哪裡」。
- `strengths`：3-5條，必須是相對於本月競品資料能站得住腳的優勢，不要寫空泛的好話
- `weaknesses`：3-5條，誠實列出，特別留意定價、聲量、第三方驗證等競品有但PowerHero沒有的部分
- `recommendations`：3-5條具體建議，每條建議要能對應到某個劣勢或某個競品的做法
- 這個分析每月都要重新檢視，不要直接複製上月內容；如果情況沒變化，也要重新確認一次再寫，而不是預設沿用

### 五大指標 / 變化紀錄
維持先前邏輯，詳見下方「五大追蹤指標」章節。

## 五大追蹤指標（每次分析依此架構查找）

> 财务類指標（市佔率、營收、LTV/AOV）為私人企業未公開資訊，無公開來源可查，
> 因此不列入追蹤範圍。以下五項皆為可從公開資料合理查找/推論的指標。

### 1. 市場與品牌定位
- 品牌定位/差異化主張：從官網「關於我們」、品牌slogan、代言人策略判讀
- 價格定位：依產品與服務指標查到的價格，歸類為「高價/中高價/中價/平價」
- 品牌知名度（聲量代理指標）：用社群粉絲數、Google搜尋結果數量、論壇提及頻率作為代理判斷依據，分類為「高/中/中低/低」

### 2. 行銷與社群表現
- 網路聲量：搜尋 Dcard、PTT（男性保養版/男孩版）近1-3個月該品牌討論篇數與熱度
- 粉絲互動率：查官方 FB/IG 粉絲數，若貼文公開可見，觀察讚數/留言數量級；查不到則寫「未公開，待查」
- 近期趨勢：與上次掃描比較，判斷聲量是「上升/持平/下降」

### 3. 產品與服務指標 ⚠️ 本項為「商品層級」，與其他四項不同
- 其他四項指標都是「一個品牌一行」，但這項是**「一個商品一行」**——同一品牌若有多款相關商品，要列出多行
- 必須同時納入 **PowerHero 自己的商品**（標記為「PowerHero（我方）」），不能只列競品，否則無法做到逐商品比較
- 每一行包含：商品名稱、劑型/規格、主成分與濃度訴求、定價、比較基準（規格說明）
- 查找時請聚焦在「跟 PowerHero 主力商品同類型」的競品商品上：
  - PowerHero 的「L-精胺酸祕魯黑瑪卡膠囊」→ 對應查九五之丹「至尊黑瑪卡」「飛龍精胺酸」、嚴萃「威猛戰神黑瑪卡」、UNIQMAN「紅瑪卡膠囊」
  - PowerHero 的「久倍戰力勁黑瑪卡粉包」→ 對應查大研生醫「精氣神瑪卡粉包」、武倍對策、UNIQMAN「極戰黑瑪卡粉」（這些都是粉包劑型，是更直接的競品）
- 如果某商品目前查不到明確單盒定價，price欄位寫「待確認」，不要編造數字，並在priceBasis說明原因

### 4. 受眾
- 目標客群輪廓：從官網文案語氣、代言人形象、產品命名推論訴求對象
- 通路布局：列出該品牌目前上架的電商平台與實體通路（官網/momo/PChome/蝦皮/屈臣氏/康是美/全聯等）

### 5. 成長動能指標
- 促銷活動頻率：查該品牌近1-2個月在momo/PChome/官網的檔期活動數量，分類「高/中/低」
- 新通路擴張：是否有新上架平台、新產品線、新合作通路
- 聲量成長趨勢：搭配指標2的論壇討論度變化，判斷「上升/持平/下降」
